Klassisk webutvikling

Rein HTML/CSS/JavaScript kan gjere alt, det er berre litt tungvindt å lage moderne nettsider utan meir drahjelp.

  • HTML blir ofte modularisert og skrive som komponentar som blir gjenbrukt ulike stadar på nettsida ved hjelp av JavaScript.
  • CSS er ofte erstatta av til dømes utility-classes eller SCSS, som brukar eit scriptspråk til å generere normal CSS som nettlesaren forstår.
  • JavaScript er ofte skrive i eigne miljø med forenklingar og forbetringar, som så blir kompilert til standard JavaScript som nettlesarane forstår.
